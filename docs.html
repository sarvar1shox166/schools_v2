awardXp() Funksiyasi — Ichki Ishlash

  1. student_xp jadvalidan joriy xp, daraja, streak, last_xp_date olinadi
  2. Streak hisoblash: bugun va kecha solishtiriladi
  3. xp += amount, daraja yangilanadi (floor(xp/200)+1)
  4. student_xp jadvaliga yoziladi (ON CONFLICT UPDATE)
  5. checkAchievements() — yangi yutuq bormi tekshiriladi
  6. { xp, level, streak, newAchievements } qaytariladi → frontend toast ko'rsatadi
